# 🚗💦 האחים מכונית בע"מ – אתר קביעת תורים

זהו אתר אינטרנט פשוט ונוח לקביעת תורים לשטיפת רכבים.  
האתר מאפשר ללקוחות לבחור **תאריך ושעה**, בודק זמינות בזמן אמת מול **Google Calendar**, שומר את התור ביומן, ולאחר מכן פותח **WhatsApp** עם פרטי ההזמנה.

🌐 **האתר פעיל (GitHub Pages):**  
https://tomerdaabul-ship-it.github.io/brotherscarwash.com/

- 🕒 בחירת **שעה פנויה בלבד**

## ⚙️ איך האתר עובד?

1. הלקוח בוחר תאריך ושעה
2. האתר בודק מול Google Calendar אילו שעות תפוסות
3. אם השעה פנויה – נוצר אירוע ביומן Google
4. לאחר הצלחה – נפתח WhatsApp עם פרטי ההזמנה

---

## 🗓️ הגדרת Google Calendar

1. צור יומן חדש ב־Google Calendar (מומלץ יומן ייעודי לעסק)
2. העתק את **Calendar ID**
3. ניתן גם להשתמש ב־`primary` (היומן הראשי)

---

## 🧩 הגדרת Google Apps Script

1. היכנס ל־https://script.google.com
2. צור פרויקט חדש
3. הדבק את הקוד מהקובץ:
   ```
   CarwashCalendarWebApp.gs
   ```
4. בתחילת הקובץ עדכן:

```js
const CALENDAR_ID = 'CALENDAR_ID_שלך';
const SECRET_TOKEN = 'טוקן_סודי_ארוך';
```

5. הגדר Timezone ל־`Asia/Jerusalem`
